# UI Kit — Landing page

A marketing landing page for EDUCA EDTECH Group, demonstrating the visual system in a real product context: dark hero with gradient line, alternating light/dark sections, the four B2C/B2B/EdTech/Editorial pillars, gradient CTA, and standard footer.

## Components

- `Header.jsx` — sticky top nav with logo, links, primary CTA pill
- `Hero.jsx` — dark `#202020` hero, gradient line, large H1, parallelogram image stack
- `ValueGrid.jsx` — four-up icon-led value cards
- `PillarsSection.jsx` — institution-pillar cards
- `StatStrip.jsx` — gradient-bg stat row
- `Quote.jsx` — light-gray section with big Rubik quote
- `CTASection.jsx` — full-bleed gradient CTA
- `Footer.jsx` — dark footer with logo, links, fine print

## Files

- `index.html` — assembled page
- `*.jsx` — individual components, loaded via Babel

Open `index.html` to view.
