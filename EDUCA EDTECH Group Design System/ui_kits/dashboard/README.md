# UI Kit — Dashboard (EDUCA LXP-style)

Internal learning-experience dashboard for the EDUCA LXP platform context: dark sidebar, light content area, course cards, progress, AI recommendations.

This is an interpretive recreation — no live screenshots of EDUCA LXP were provided. It applies the EDUCA EDTECH Group visual system (gradient accents, parallelogram motifs, teal line icons, pill buttons, underline inputs) to a typical course-progress dashboard.

## Components

- `Sidebar.jsx` — `#202020` left rail with logo, nav, gradient line accent
- `TopBar.jsx` — search, user avatar, notifications
- `WelcomeHero.jsx` — gradient banner with greeting
- `CourseCards.jsx` — three course cards with progress bars
- `AIRecommendation.jsx` — feature card highlighting AI-driven recommendations

## Files

- `index.html` — assembled dashboard view
